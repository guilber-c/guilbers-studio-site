Quick Overview of TidyQuant

tidyquant is an R package that integrates quantitative financial analysis with the tidyverse, it pulls financial/economic data via quantmod, TTR and xst and let us work with it using tidy, pipeable workflows for charting, technical analysis and portfolio/performance analytics.

1. Install

install.packages("tidyquant")

2. Load it

library(tidyquant)

This also loads core tidyverse pieces (dplyr, ggplot2, tidyr, purrr, lubridate) plus xts, quantmod, TTR, PerformanceAnalytics.

2. Vectors & element-wise operations

Well, what is a vector? In mathematics, a vector is an object that has both magnitude (size/length) and direction. It’s typically represented as an arrow, or numerically as an ordered list (tuple) of numbers called components or coordinates.

In R this has more to do with an ordered list, using the same tibble that we use before sp500$symbol is just a numeric vector for R.

Looks simple but I will try to understand a little better.

I will take mean() function. In math, mean() is a function \(f: \mathbb{R}^n \to \mathbb{R}\) it takes a vector of \(n\) numbers and maps it to a single number (a scalar).

The rule is: \[ f(x_1, x_2, \ldots, x_n) = \frac{1}{n}\sum_{i=1}^{n} x_i \]

So we have one input (a vector), one deterministic rule (sum, then divide), one output (a scalar). We can look at a practical example using the vector \(x = (1,2,3,4,5,6)\)
